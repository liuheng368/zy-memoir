# source

本目录为 **feature 素材占位**：可放置**线框与页面截图**（供 **`spec.md`** §1 嵌入）、设计稿、流程图、抓包、会议纪要等**研发辅助**文件。与 **`spec.md`**（需求）、**`plan.md`**（技术方案、**AC**、UI 与 Token）、同级的 **[`prd/`](../prd/)**（**上游 PRD / 产品方案**原文留存）分工：**需求**以 **`spec.md`** 为准，**验收标准（AC）** 默认以 **`plan.md`** 为准。

（由 **`_shared/create-feature-spec`** 脚本初始化；**spec-creator** Skill 会调用该脚本；可按需增删文件或删除本说明。）
