| 子 | 操作 | 预期 | 结果 |

| 17.4 | 游客顶栏右上 | **无** ⚙ 管理员入口 | [ ] |
| 17.5 | 游客点徽标 / 气泡内「去登录」 | 弹学生登录浮层；输入正确名单后升级为 `student`，编辑入口解锁 | [ ] |
| 17.6 | 游客访问 `/admin?token=<错>` | 跳 `/login` 浮层（不自动转为游客直通管理页） | [ ] |
| 17.7 | **游客控制台直调写接口**（见下方 AC-17.7 注） | 全部返回 `{ ok:false, code:'UNAUTHORIZED' \| 'FORBIDDEN' }`，数据库无变更 | [ ] |
| 17.8 | sessionStorage `zy-guest-tip-dismissed` 关 popover 后本会话不再弹 | 刷新页面（同会话）不再弹气泡 | [ ] |

> **AC-17.7 控制台直调脚本**（在浏览器 DevTools Console 粘贴）：
>
> ```js
> // 游客态下手工调用任一写云函数，应被云函数侧 401 / 403 拒绝
> import('@cloudbase/js-sdk').then(async ({ default: cloudbase }) => {
>   const app = cloudbase.init({ env: 'zy-memoir-d5gaxbvyxe80564f4' })
>   // 注意：游客没走匿名登录时 callFunction 本身会被 SDK 报"未登录"，
>   // 所以下方 token 故意造个空值，验证云函数侧的 verifyToken 兜底
>   const r = await app.callFunction({
>     name: 'updateStudentIntro',
>     data: { token: '', intro: '哈哈我入侵啦' },
>   })
>   console.table(r.result)  // 期望 { ok:false, code:'UNAUTHORIZED', ... }
> })
> ```
