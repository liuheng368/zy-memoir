## description

为钟园幼儿园 2024 届大二班制作一个可用于记录班级同学照片、录音等信息的网站。

## 场景

1. 手机端
2. PC 端

## 权限

- 学生（只可编辑自己的信息）
- 老师（只可编辑自己的信息）
- 管理员（可以对所有可编辑信息进行上传、删除操作）
- **游客**（**仅可浏览**全站内容，**禁止**任何编辑 / 上传 / 删除）

**不同权限根据地址路由进行区分**

## 核心页面

**默认强制登录策略**：除 **管理员路由**（如 `/admin`）以外，**任何其他方式**打开页面都会**优先弹出登录浮层**；浮层支持「跳过登录」直接以**游客态**进入。游客态下整站可浏览但**无法编辑**。

**登录浮层不再做角色 Tab 选择，按 URL 直接区分**：

| 访问 URL | 弹出 / 跳转形态 | 角色边界 |
| -------- | -------------- | -------- |
| `/` | 学生登录浮层（**学号 + 姓名**） | 学生 |
| `/teacher` | 老师登录浮层（**单选老师姓名**） | 老师 |
| `/admin?token=xxx` | **不弹浮层**，token 校验通过即进入管理员态 | 管理员 |
| 任意非管理员 URL + 跳过登录 | 浮层底部「↪ 跳过登录，仅浏览」 | 游客 |

### 1. 用户登录（浮层形式）

- **a. 学生**（路由 `/`）：无需注册，**学号 + 姓名**即可登录。名单 JSON：文件名 **`kindergarten_students_en_keys.json`**，需要云服务建表。
- **b. 老师**（路由 `/teacher`）：3 位老师，选择自己的名称进行登录。
- **c. 管理员**（路由 `/admin?token=xxx`）：无需登录，通过**专属管理员路由**直达，不进入登录浮层。
- **d. 游客（跳过登录）**：学生 / 老师两种登录浮层底部均提供「**跳过登录，仅浏览**」入口；点击后进入主页**游客态**，与已登录用户**展示一致**，但所有「编辑 / 上传 / 删除 / + / 换头像」入口**统一隐藏**，长按等编辑手势失效。
- **学生 / 老师浮层完全独立、不互通**：浮层内**不**提供「学生 → 老师」/「老师 → 学生」切换入口；用户若误打开错误 URL，需自行修改地址栏到正确路径，或重新点击正确的角色链接。**配套约定**：班级链接发放时按角色发放（家长群发 `/`、老师群发 `/teacher`），从源头规避误打开。

### 2. 主页

展示元素：

- **a.** 多张合影，轮播展示。
- **b.** 展示老师信息：头像照片，几段录音（点击可播放）。
- **c.** 所有学生的头像墙（按学号排序），但不展示学号，可点击。**设计要求：不拘泥于常规格子布局**。

### 3. 学生页（浮层形式）

分：**主、客态**（**游客态**与客态展示完全一致）

- **主态**（仅当前学生本人）：
  - 名称不可修改。
  - 编辑其他信息：一段 **300 字** 自我介绍。
  - 上传、删除：**头像、3 张照片、5 段录音**。
- **客态**（其他登录用户 / 老师 / 管理员 / **游客**）：
  - 查看头像、照片，听录音；**所有编辑入口隐藏**。

### 4. 教师页（浮层形式）

- **主态**（仅当前老师本人）：上传、删除头像照片、多段录音。
- **客态 / 游客态**（如开放查看）：仅查看头像与播放录音；**无任何编辑入口**。

### 5. 管理员页

- 上传、删除：主页合影。
- **入口**在主页右上角，**仅管理员**可见；**游客态**下不展示。

## 主要功能说明

### 1. 图片上传

- 手机端：系统相册、相机。
- PC 端：本地图片上传。
- 格式：**png / jpg / webp** 等常见图片格式，**不包含 livephoto**。
- 压缩：**单张图片要求压缩到 3M 以下**。

### 2. 语音上传

- 时长：**60 秒**。
- 保存格式：**MP3**。

### 3. 语音播放

- 点击后直接进行播放。

## 技术栈

- 前端：**Vue**
- 后端服务：**CloudBase 云服务**
- 存储：**CloudBase COS**
